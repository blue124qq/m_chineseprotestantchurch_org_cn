初信基石

2016-01-31 16:20:21   作者：New Believer Bible   来源：New Believer Bible

引言：谁是真神？ 谁是耶稣？ 谁是圣灵？ 谁是恶魔？ 谁是天使？谁是鬼灵？ 天堂何在？ 地狱何在？……

一. 谁是真神？

　　几千年前埃及的统治者，法老拋出今人仍发问的问题：「耶和华是谁，使我听他的话呢？」这是一个很好的问题，却不是一个容易应对的课题。它如此困难，是因为我们要用有限的头脑去了解这位无限、永恒的神。
　
有句话说：「神若只有你的脑子那么小，他就应付不了你那么大的需求。」为此，你若不能完全了解神，和他所行的某行事，你也不需要感到恼怒。圣经应许我们：「我如今所知道的有限；到那时就全知道，如同主知道我一样。」(林前十三12)
　
如今，我们可以从神的话语中认识神。

　　1. 神是无所不知、无所不在，和无所不能的：创造宇宙万物的主宰亲切的认识他一切的造物。(诗一三九)
　　2. 神是圣洁的：神无可比拟的圣洁值得我们的敬拜。(赛六1-8)
　　3. 神是慈爱和公义的：神的公义显明在慈爱中；他法外施恩。(彼后三3-9)
　　4. 神是有位格的：神独特的属性使他有别于其它宗教信仰的神。(徒十七22-31)
　　5. 神在掌权：认识神仍在掌管是重要，即使四周的事况显得混乱时。(哈三3-17)

　　圣经中的神是独一的真神：在许多人强调有多神存在时,唯有圣经中的神是又真又活,值得敬拜的. (赛四十五9-23)

二. 谁是耶稣？

　　在历史上有许多人尝试回答这个问题。许多人找到正确的答案，许多人没有。我们回答这个问题最好的资源，是来自神自己的话语。

　　圣经提供关乎耶稣不可忽视的真理，且要求我们给予响应。

　　任何严肃研读圣经，寻求明白耶稣的人，都必须回答两个重要的问题：
　
第一，你想耶稣基督是谁？第二，他是谁？

　　路易斯(C. S. Lewis)曾经如此查证：「你必须作出抉择。这个人是神的儿子，或只是一个狂人。你可以当他是傻瓜，也可以唾弃和杀害他如同被鬼附的人。」
　
或者，你也可以俯伏在他面前，称他为主为神。只是，请你不要当他是一位伟大的人类导师，因他不是，他也没有如此的意愿。」耶稣不只是好人，他更是神人。让我们从圣经中查证他是谁。

　　1. 耶稣的人性：耶稣道成肉身成为神为人的最高榜样。(腓二5-11)
　　2. 耶稣的神性：耶稣虽然身为人，他仍然是神。(西一15-20)
　　3. 耶稣的使命：耶稣降世特定的使命是要拯救人类脱离罪恶。(路四16-21)
　　4. 耶稣的牺性：耶稣承担极大的十架之苦，好叫我们与他同享永生。(赛五十三)
　　5. 耶稣的权能：耶稣能改造一个不可能的人， 变成大有能力的见证人。(徒四13)
　　6. 耶稣的统管：耶稣永恒的国度超越时空的界限。(启一4-8)

三. 谁是圣灵？

　　圣灵是父子灵，三一真神中最奥秘的一位。许多人混淆于神有三个位格的观念。坦白说，我们实在不能完全了解这属天的奥秘。有些人甚至的错误认为圣灵是「它」而不是「他」，可能是因为圣经描述圣灵好象风，或是好象鸽一样临在耶稣身上。

　　其实，耶稣形容自己是「生命的粮」，父神也形容自己是「避难所」。其实，如此的描述是要帮助我们了解神的特性。当然如此独特的形容也不是指圣灵只是一种「能力」。耶稣形容圣灵是真理的灵，「他要引导你们明白一切的真理...并要把将来的事告诉你们。」(约十六13)
　
圣灵有着个别的位格，他在耶稣基督的跟随者身上也有具体的工作。

　　试从圣经了解他：
　
　　1. 圣灵帮助谁？圣灵加力量和充满基督的跟随者。(徒二1－40
　　2. 圣灵如何与圣父和圣子同工？圣灵配合帮助我们过讨神喜悦的生活。(彼前一2)
　　3. 为什么神赐给我们圣灵？圣灵成为我们属神的印记。(弗一13－14)
　　4. 圣灵如何在我们生命中工作？圣灵要引领我们亲近神，重生我们和加我们力量。(约十四15－17)
　　5. 我们何时会得罪圣灵？我们会在六种具体的情况下得罪圣灵。(徒五1－10)
　　6. 为什么基督徒需要圣灵？因为没有圣灵帮助，无法为主而活。(加五16－26)

四. 谁是恶魔？

　　恶魔的样子像什么？是不是像卡通画中披红袍，手拿长柄叉坐镇地狱宝座的？或是遍行全地以光明天使样貌出现的？很可惜，许多人对恶魔没有正确的认识。许多人低估牠和牠的智勇，甚至还疑惑牠的存在呢！

　　有人问布道家芬尼：「你是否相信恶魔确实的存在？」芬尼回答说：「你试试正面对抗牠一阵子，你就知道牠是否确实存在。」其实当你委身耶稣基督时，你就会感受到恶魔确实存在。

　　圣经已经向我们清楚显示，恶魔是如此的活跃暗中活动。同时，圣经也告诉我们恶魔和限制和终结。当我们更多了解这聪明精灵的诡计，我们就更能装备自已，谨防牠的攻击。

　　下述几点陈明恶魔，也称为撒但的真面目：

　　1. 撒但本源自何处？撒但因为骄傲由天上堕落。(结廿八12-19)
　　2. 撒但有什么能力？撒但确实有能力涉及世上某些事物。(林后四3-4)
　　3. 撒但有什么限制？我们不当低估撒但的能力，然而牠的能力有必然的限制。(伯一1-二10)
　　4. 撒但如何攻击人？撒但精于混淆和歪曲真理好欺骗人。(创三1-7)
　　5. 谁能挫败撒但的意图？那些不惜生命为基督摆上的弟兄必要得胜仇敌恶魔。(启十二10-12)

五. 谁是天使？

　　大部份人都相信天使的存在？圣经提供天使和他活动最可靠的资料。然而，天使的神秘面目到底是怎样的？他们是否穿著白色长袍，有着鸟儿般大翅膀？他们存在的目的是什么？

　　你可以称天使为“神的密探”，他们是专属于神的，大部份时间是隐藏的，在掩护下工作，惟有在特别的情况下，神才允准人们看见他。因神知道，若是人随时可以看见天使，就会将他们当偶像来拜。

　　让我们看看圣经如何讲说天使和他们在世上所扮演的角色。

　　1. 为什么神创造天使？神创造天使成为独特的灵界活物，为要敬拜基督，关照他的跟随者。(来1：4-14)
　　2. 天使在基督徒生命中做了什么？神吩咐天使保护基督徒免受伤害。(诗91：1-16)
　　3. 天使如何参与祷告？我们的祷告会引发灵界里天使和恶魔的争战。(但10：1-11)
　　4. 天使在世界的末了将会扮演什么角色？天使在传播永远的福音上将扮演关键性的角色。(启14：6-7)

六. 谁是鬼灵？

　　正如好的天使看顾你；坏天使，就是鬼灵想使你灭亡。圣经告诉我们撒但从天上堕落后，率领了三分之一的天使跟随牠背叛神(启十二4)。

　　虽然我们不知道确实的数目，但必然成千上万(路二13；太廿六53)。这些堕落的天使也被称为污鬼邪灵；牠们在撒但高度的组织，统领下完成撒但的使命，正是耶稣所讲的，偷窃、杀害和毁灭。

　　神己经在圣经中提示鬼灵的行径，虽然不是那么详细，但已经足够我们认识牠们，且无须自行进入灵界寻求真相，以免落在魔鬼的陷阱中，成为魔鬼的奴力。

　　1. 魔鬼相信什么？奇怪的是，牠们竟然相信一的真神。(雅一19)
　　2. 魔鬼会伤害我们吗？那些与基督有美好关系的人，必不受魔鬼的侵害。不信的或招鬼的人，或有这种危险。(徒十九13－20)
　　3. 什么会使魔鬼失去能力？跟随耶稣的人奉主名的时候，魔鬼就要战抖。(路十1－20)


七. 天堂何在？

　　圣经所描述的天堂是如此奇妙，栩栩如生黄金街碧玉城、无病无惧无伤痛。
　
虽是如此我们仍然无法想象天堂这幅「大图画」。那是因为我们实在难以领会天堂的完美和荣耀。除却光彩壮丽之外，真正使天堂景象如此壮观的是我们将永远与神同在。

　　正如诗人所表达的：「你必将生命的道路指示我；在你面前有满足的喜乐，在你右手中有永远的福乐。」
　
虽然在世上的我们，未能完全解答关乎天堂的疑难。但是圣经实在响应了一些我们最想深入探讨的问题。

　　1. 什么人得以进入天堂？天堂是为相信耶稣是主的人预备的。(约十四1－6)
　　2. 什么时候基督徒得以进入天堂？基督徒呼出地上最后一口气之后，就吸进天上的第一口气。(林后五6－9)
　　3. 我们在天堂仍能认出彼此么？当我们天上的形体和地上的形体结合时，我们就会类似天使一样。(太廿二23－33)
　　4. 天堂的生活将是怎样的？我们不再需要为地上生命劳碌，我们会愉快的与天父生活在一起。(启七13－17)

八. 地狱何在？

　　根据圣经，我们死后的生命有两种选择：天堂或是地狱。目前，越来越多人相信地狱的存在，但却不相信自己正走向地狱，他们相信地狱是为那些社会中邪恶的罪犯，或相关因素预设的。

　　然而，神是根据不同的标准审判世人。正如神裁决你下地狱不是因为你是的恶人，神允准你上天堂也不是因为你是好人。而是，我们都是罪人，本来都应该永远在地狱里(罗三22－23)。

　　神清楚的说道，那些拒绝神透过独生儿子耶稣基督成就救恩的人，他们的永恒的去处是在地狱。
　
神重复的给每个人有充足的机会选择生命——世上更丰盛的生命，天堂永恒的生命(彼后三9)。
　
若你还没有作出抉择，或是想知道不信主的人所面对地狱是怎样的，可参看下述的经文。但请把握机会作出生命的抉择。

　　1. 地狱是怎样的？地狱是一个永不止息，隔绝痛苦的地方。(路十六19－31)
　　2. 谁将会去地狱？那些名字没有记在生命册上的人，至终将被押送到地狱遭受永恒的刑罚。(启廿11－15)
　　3. 地狱中最惨的刑罚是什么？那是永远与神隔绝分开。(帖后一7－10)

九. 爱神爱人

　　有一次，耶稣被人问及：「诫命中那是第一要紧的呢？」耶稣回答说：「以色列阿！你要听，主我们神，是独一的主。你要尽心、尽性、尽意、尽力爱主你的神。」其次就是说：「要爱人如己。」这是两条最要紧的诫命，因为你若真正爱神，就会乐意遵行讨神喜悦的事。

　　同时，若你真正爱人如同自己，你就会关爱他人的福祉。当然，你会爱神是因为知道神先爱你(约壹四19)。圣经说：在我们还作罪人的时候，神就透过基督为我们舍命在十字架，如此显明了神对我们的爱(罗五8)。我们更多了解这些奇妙的真理，我们对神的爱就会更加滋长。

　　1. 我们生命中最大的爱应当属于神。我们应当全然爱神，和明白他对我们的爱，才能全然彼此相爱。(申六4－5)
　　2. 我们行事和对他人的爱，应以基督的爱为榜样和标准。(弗五1－2)
　　3. 爱超越所有的属灵恩赐。基督徒明白真爱的意义，若生命能表彰这份爱就是最大的见证了(林前十三1－13)。
　　4. 我们对神爱的深度将直接影响我们服事神和人的能力(约廿一15－17)。
　　5. 我们对人的爱反映内心状况，也表彰活出基督徒生活的能耐(约壹二9－11)。
　　6. 我们爱神和爱人的心应当同时越发增长充足(帖前三12－13)

十. 彼此饶恕

　　基督徒生命中其中一个重要的原则是饶恕。耶稣在十字架上为钉他的仇敌祷告，求神赦免他们的无知，显明了饶恕的真理。(路廿三34)

　　他的言语是那么的有力和出其不意，导致钉在身旁的强盗悔改。因为耶稣完全的赦免我们。他要我们效法他的榜样，饶恕他人。
　
正如圣经说：「并要以恩慈相待，存怜悯的心，彼此饶恕，正如神在基督里饶恕了你们一样。所以你该效法神，好象蒙慈爱的儿女一样。」(弗四32－五1)

　　圣经提示我们几种饶恕的特性：

　　1. 饶恕根源自神的赦免。我们对人的饶恕乃流露自神对我们的赦免。(可十一25)
　　2. 饶恕是没有止尽的。做为一个基督徒，没有一种错误是太大或太小不能饶恕的。(太十八21－35)
　　3. 饶恕是不可选择的。你不能选择饶恕某些人，不饶恕某些人。(太五43－48)

　　饶恕能拆除分隔的墙。当你选择饶恕时，你将经验到真正的自由。(创四十五1－15)

十一. 追求圣洁

　　圣洁，在这时代越来越少听到的品质。当我们听到圣洁时，通常是与性爱课题相关。
　
然而，圣洁超越这一切，它包括洁净的思想，渴望做对的事，立志遵行神的话语。耶稣同样提示圣洁的重要性，他应许：清心的必得见神的面(太五8)。
　
耶稣用“心”这个用词表达我们意志、情感和思想进程的中枢，必须远离罪恶。

　　下述经文显示圣经中的圣洁观，以及对跟随耶稣基督的我们的影响。

　　1. 不要让自己置身不必要的试探：知道自己德性上的软弱，避免自陷不能抗拒被罪诱惑的境况。(撒下十一1－十二14)
　　2. 保守自己的思想：不要让世俗不道德的思绪充满你的思想。(提后二22)
　　3. 提防内心的罪行：一个内心犯奸淫的人所有的罪疚，如同真正犯过的人。(太五27－30)
　　4. 避免淫乱的关系：神特别警戒那些活在不道德关系中的人。(帖前四1－8)
　　5. 若你跌倒，祈求神赦免你，洁净你的心和欲望：惟有神能赦免你恢复你的喜乐，使你的心充满正确的愿望。(诗五十一7－15)
　　6. 保持永恒的观点：不要浪费你的时间追求世间短暂的快乐。(彼后三10－11)
　　7. 生活讨主喜悦：让你的生活降服在圣灵的带领之下，他是唯一可以帮助过一个讨主喜悦的生活。(罗八5－8)

十二. 坚持到底

　　当你灵性低潮，情绪低落时，你可能以为神不再理你。又或许因为看到某些著名的基督徒跌例，你灰心丧志，你想或许你是魔鬼名单中下一个击打的对象。
　
其实，神不但不会让你受试探过于你所能受的；他更加要透过试探帮助你更加的坚强。
　
圣经用马拉松竞赛，说明在人生长途中，虽会面对不可避免的挣扎，我们必须调整步伐，坚持到底的跑完全程。

　　1. 坚持到底必有成果：当你在认识神的话语上稳健成长，并能应用在生活上，你就能赢取他人归主。(路八15)
　　2. 生命中面对的试探将使你更加刚强：莫将困难视为信仰的拦阻，乃将它视为灵性成长的良机。
　　3. 基督也曾为我们忍受诸般的痛苦：耶稣受苦的榜样，必要激励我们在生命竞赛中持守刚强的信心。
　　4. 神尊荣坚持到底的人：全心委身给神，将使你的信心由始至终完好无损。(书十四6－14)

十三. 诚实廉正

　　诚实廉正的事在这时代越发少见。然而圣经告诉我们，这是敬虔男女生命质素中不可缺少的一部份。然而世界却趋向于掩饰或曲解基督徒基督徒生命的观点，它们将基督徒归类于与现实难以融合的一群。其实，基督徒是一群容让神影响他们生命的各领域，并付诸于行在日常处事待人，包括处理事业、财务和人际关际等事工。

　　我们可从下述经文中看出，我们的生命实在需要充满诚实廉正：

　　1. 我们应当超越论断：但以理的信心和诚实，使他与众不同。(但六3－4)
　　2. 我们的行为应当使人归荣耀于基督：在四围没有信主的人群中，活出美善和诚实的生命，必使他人因你的好行为归荣耀于神。(彼前二9－12；太五16)
　　3. 我们应当成为别人的榜样：我们应当严肃的实践诚实廉正的生活，好成为后进，或其它基督徒的榜样。(多二6－8)
